### Admin Dashboard
## Overview

This is a modern Admin Dashboard web application built with HTML and CSS. The project is designed to demonstrate the power of CSS Grid and Flexbox for creating responsive, modular, and professional layouts.

The dashboard includes:

A fixed sidebar with navigation links
A header with search functionality, notifications, and user info
Overview cards with statistics
Analytics widgets for data visualization
Responsive grid layout using modern CSS techniques


## Features
Responsive layout using CSS Grid (auto-fit, minmax())
Modern UI/UX with hover effects, shadows, and clean typography
Sidebar navigation for easy access to different sections
Header with search and user profile
Overview cards for key metrics
Analytics section with placeholder widgets
Installation

Clone this repository:

git clone https://github.com/youssouoph2008/admin-dashboard.git
Open index.html in your browser.
## Usage
Modify the HTML to add your own content and sections.
Customize the CSS for your own colors, fonts, and layout.
Replace placeholder content with real data or charts.
## Technologies Used
HTML5
CSS3 (Grid, Flexbox, minmax, clamp)
